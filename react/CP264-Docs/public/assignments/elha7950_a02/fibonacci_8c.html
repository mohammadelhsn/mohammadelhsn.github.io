int dpbu_fibonacci ( int * f,
int n )

Compute and return the nth Fibonacci number F(n) using dynamic programming bottom-up method with external array f[n+1] of initial value -1 for all elements. Namely it fills up f[] byf[0]=0, f[1]=1, a[i]= a[i-2]+a[i-1] for i=2,...,n, and return f[n].

- the nth Fibonacci number

◆ dptd_fibonacci()

int dptd_fibonacci ( int * f,
int n )

Compute and return the nth Fibonacci number F(n) using dynamic programming top-down method with external array f[n+1] of initial value -1 for all elements. Namely it fills up f[n+1] by recursion function call. Specifically it returns f[n] if f[n]>0 else sets f[n] = dptd_fibonacci(f, n-2) + dptd_fibonacci(f, n-1) and then returns f[n]

n- the n for F(n)
Returns
- the nth Fibonacci number F(n).
