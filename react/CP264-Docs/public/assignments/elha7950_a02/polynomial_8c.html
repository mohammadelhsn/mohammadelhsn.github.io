◆ derivative()

void derivative ( float * p,
float * d,
int n )

Compute the derivative of input (n-1)-th degree polynomial p(x) = p[0]*x^{n-1} + p[1]*x^{n-2} +...+ p[n-2]*x^1 + p[n-1]*x^0 output the coefficients of the derivative polynomial in an array. The derivative of the above polynomial p(x) is as follows. p'(x) = (n-1)* p[0]*x^{n-2} + (n-2)*p[1]*x^{n-3} +...+ p[n-2]*x^0

Parameters
*p- pointer to coefficient array of input polynomial.
*d- pointer to coefficient array of output polynomial.
n- the number of coefficients of the input polynomial
