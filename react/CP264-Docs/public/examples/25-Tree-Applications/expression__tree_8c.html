Recursively frees all nodes in a binary tree and sets the root pointer to NULL.

◆ evalulate_expression_tree()

int evalulate_expression_tree ( tnode * root)

Evaluates the value of an arithmetic expression represented by a binary expression tree.

Parameters
rootPointer to the root node of the expression tree.
Returns
The integer result of evaluating the expression.

The function assumes that the tree nodes represent either operators (such as '+', '-', '*', '/') or integer operands. It recursively evaluates the left and right subtrees and applies the operator at each internal node.
