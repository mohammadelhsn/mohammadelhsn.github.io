void heap_sort ( int a[],
int n )

Sorts an array of integers in ascending order using heap sort algorithm.

Parameters
aThe array to be sorted.
nThe number of elements in the array.

The function first builds a max-heap from the array, then repeatedly extracts the maximum element (root of the heap) and places it at the end of the array, shrinking the heap size each time. The process continues until the array is sorted.
