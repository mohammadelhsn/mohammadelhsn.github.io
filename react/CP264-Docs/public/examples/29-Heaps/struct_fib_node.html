Detailed Description

Fibonacci Heap Node structure.

key {int} - The key or value stored in the node. degree {int} - Number of children of the node. parent {*FibNode} - Pointer to the parent node. child {*FibNode} - Pointer to one of the node's children. left {*FibNode} - Pointer to the left sibling node in the circular doubly linked list. right {*FibNode} - Pointer to the right sibling node in the circular doubly linked list. marked {int} - Flag indicating whether the node has lost a child since it became a child of its current parent (used in cascading cuts).
