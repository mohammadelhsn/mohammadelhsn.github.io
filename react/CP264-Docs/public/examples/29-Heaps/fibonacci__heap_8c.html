void consolidate ( FibHeap * heap)

Consolidate the root list of the Fibonacci heap to merge trees of the same degree.

Parameters
heapPointer to the Fibonacci heap to be consolidated.

This function merges roots with the same degree until at most one root of each degree remains. It reorganizes the root list and updates the pointer to the minimum node. Consolidation helps maintain the amortized time complexity of heap operations.
