typedef struct stnode snode

Stack node structure used to store pointers to heap nodes.

This structure is used in the heapify-up operation to keep track of the path from the root node to the current node in the heap.

Parameters
tnpPointer to a tree node (tnode) in the heap.
nextPointer to the next stack node in the stack.

◆ tnode

typedef struct btnode tnode

Node structure for a linked binary heap (complete binary tree).

key {int} - The priority or key value of the node. data {int} - The associated data stored in the node. left {*tnode} - Pointer to the left child node. right {*tnode} - Pointer to the right child node.

◆ delete()

void delete ( tnode ** rootp,
int * count )

Delete the root node (minimum or maximum, depending on heap type) from the linked binary heap.

This function removes the root node from the heap rooted at *rootp, maintaining the complete binary tree property and the heap order property. The count parameter keeps track of the number of nodes and is updated accordingly.

Parameters
rootpPointer to the root pointer of the heap (so root can be updated).
countPointer to the integer holding the current number of nodes in the heap.

Insert a new node into a linked binary heap represented as a complete binary tree.
