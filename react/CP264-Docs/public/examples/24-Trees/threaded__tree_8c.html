typedef struct tree_node tnode

Define node structure of a threaded binary tree.

data {int} - data field of tree node thread {int} - flag indicating if right pointer is a thread (1) or a child (0) left {*tree_node} - pointer to the left child right {*tree_node} - pointer to the right child or threaded successor
