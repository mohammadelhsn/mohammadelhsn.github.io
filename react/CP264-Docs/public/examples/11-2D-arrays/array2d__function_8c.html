Transposes a square matrix in place using pointer arithmetic.

This function takes a pointer to the first element of an n x n matrix stored in row-major order and transposes the matrix in place. The matrix is represented as a one-dimensional array.

Parameters
nThe number of rows and columns in the square matrix.
mPointer to the first element of the matrix (int*).
Note
The function assumes that the input matrix is square (n x n). The transposition is performed in place, so the original matrix will be modified.
