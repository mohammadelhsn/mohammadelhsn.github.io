◆ clean()

void clean ( hashtable * ht)

Frees all memory used by the hash table and its elements.

This function traverses each linked list in the hash table, frees all nodes, and finally frees the array of pointers and the hash table structure itself. After calling this function, the hash table pointer becomes invalid and should not be used unless reinitialized.

Parameters
htPointer to the hash table to be cleaned and freed.

void display ( hashtable * ht)

Displays the contents of the hash table.

This function iterates over each index of the hash table and prints the key-value pairs stored in the linked list at that index. If an index has no elements, it prints nothing for that index. Useful for debugging and inspecting the internal structure of the hash table.

Inserts a new hash node into the hash table.

This function uses the hash function to compute the index, then inserts the node into the appropriate linked list, keeping the list ordered by key. If the bucket is empty, it simply places the node at the head. Otherwise, it traverses the list and inserts the node in sorted order.

◆ main()

int main ( )

Demonstrates the usage of the hash table operations: creation, insertion, search, deletion, display, and cleanup.

This function creates a new hash table, inserts multiple key-value pairs, performs a search for a specific key, deletes that key from the hash table, searches for the key again to verify deletion, displays the current state of the hash table before and after deletion, and finally cleans up all allocated memory.

Steps performed:

  1. Create a new hash table of fixed size.
  2. Insert multiple nodes with various keys and values.
  3. Display the hash table contents after insertion.
  4. Search for a specific key and print the result.
  5. Delete the node with that key.
  6. Search for the deleted key again to confirm removal.
  7. Display the hash table contents after deletion.
  8. Clean all allocated resources to avoid memory leaks.
Returns
int Returns 0 upon successful execution.
