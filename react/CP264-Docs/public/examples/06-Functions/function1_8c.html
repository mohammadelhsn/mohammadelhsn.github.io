int main ( void )

Entry point of the program demonstrating function call and global variables.

This program initializes two global variables a and b to 1 and 2 respectively. Then it calls the function add passing these variables as arguments. The function add returns the sum of its parameters plus 1, but the return value is not used in main.

The assembly output generated by compiling this program is shown in the comment block, illustrating how parameters are passed and functions are called at the assembly level.
