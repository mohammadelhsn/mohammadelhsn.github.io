int balance_factor ( tnode * root)

Calculates the balance factor of the given AVL tree node.

The balance factor is defined as the height difference between the left and right subtrees: balance_factor = height(left subtree) - height(right subtree)

◆ rotate_left()

void rotate_left ( tnode ** rootp)

Performs a left rotation on the subtree rooted at the node pointed to by rootp.

This operation is typically used in AVL trees to restore balance after insertions or deletions. After the rotation, the right child of the original root becomes the new root of the subtree, and the original root becomes the left child of the new root.

Parameters
rootpA double pointer to the root node of the subtree to be rotated.
